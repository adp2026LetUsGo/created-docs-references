## SECTION 3 — APPLICATION LAYER

### Commands (all inherit AHS.Common.Application.SignedCommand)

  RegisterShipmentCommand:
    CargoType, InsulationType, OriginLocation, DestinationLocation, PlannedDeparture, TenantId

  RecordExcursionCommand:
    ShipmentId, SensorId, ZoneId, ObservedCelsius, MinLimit, MaxLimit, Severity, TenantId

  ResolveExcursionCommand:
    ShipmentId, ExcursionEventId, ResolutionNote, TenantId

  SealShipmentCommand:
    ShipmentId, FinalStatus, QualityDecision, TenantId

  ApplyWhatIfChangeCommand:
    ShipmentId, ParameterName, PreviousValue, NewValue, TenantId

### Command Handlers (explicit injection — no MediatR)

  RegisterShipmentHandler(IShipmentRepository, IEventStore, ICellEventPublisher)
    → creates Shipment aggregate → appends to EventStore → publishes ShipmentCreated
    → returns Guid (new ShipmentId)

  RecordExcursionHandler(IShipmentRepository, IEventStore, ICellEventPublisher)
    → loads Shipment → calls RecordExcursion() → appends events → publishes

  ResolveExcursionHandler(IShipmentRepository, IEventStore, ICellEventPublisher)
    → loads Shipment → calls ResolveExcursion() → appends events → publishes

  SealShipmentHandler(IShipmentRepository, IShipmentReadRepository,
                      IEventStore, ICellEventPublisher)
    → loads Shipment + temperature readings
    → calculates MKT via MeanKineticTemperature.Calculate() (from AHS.Common)
    → calls Seal() → appends events → publishes ShipmentSealed

  ApplyWhatIfChangeHandler(IEventStore)
    → appends WhatIfParameterChanged event directly (no aggregate load needed)
    → event contains full audit trail including ReasonForChange

### Queries (Dapper — NOT EF Core)

  GetShipmentByIdQuery  → ShipmentDto?
  ListActiveShipmentsQuery → IReadOnlyList<ShipmentSummaryDto> (cursor pagination)
  GetColdChainReportQuery → ColdChainReportDto? (for GxP audit export)

### Oracle (REQ-001) — Application Service

  LogisticsOracle:
    CalculateAsync(OracleRequest req, CancellationToken ct) → ValueTask<OracleResult>
    Rules:
      - Route Threat Level:           40% weight
      - Carrier Reliability:          30% weight
      - External Temperature Forecast: 30% weight
      - Passive Insulation modifier:  +15% base penalty
      - Pessimistic TTF: PhysicalTtf × (1 - riskScore/100 × 0.60)
      - Safe window: PessimisticTtf × 0.80
      - XAI DNA: exactly 14 diagnostic factors
      - P99 target: < 10ms (use ValueTask, no LINQ, no heap allocations)

### DTOs (records, registered in JsonSerializerContext)

  ShipmentDto: all Shipment fields
  ShipmentSummaryDto: Id, CargoType, Status, InsulationType, IsSealed, TenantId
  CreateShipmentRequest: CargoType, InsulationType, OriginLocation,
                         DestinationLocation, PlannedDeparture, ReasonForChange
  SealShipmentRequest: FinalStatus, QualityDecision, ReasonForChange
  OracleRequest: RouteId, CargoType, InsulationType, RouteCategory,
                 CarrierId, CarrierReliability, CarrierIncidents12M,
                 ForecastMaxCelsius, ForecastMinCelsius, ForecastHumidityPct,
                 SetpointMinCelsius, SetpointMaxCelsius, PhysicalTtfHours
  OracleResult: RequestId, CalculatedAt, Risk (breakdown), Ttf (analysis),
                Dna (14 factors), Recommendation, RequiresImmediateAction
  ColdChainReportDto: ShipmentId, CargoType, MktCelsius, ExcursionCount,

## SECTION 6 — API LAYER

### Program.cs (Native AOT — CreateSlimBuilder)

```csharp
var builder = WebApplication.CreateSlimBuilder(args);

builder.Services.ConfigureHttpJsonOptions(o =>
    o.SerializerOptions.TypeInfoResolverChain.Insert(0, ColdChainJsonContext.Default));

builder.Services.AddAuthentication(JwtBearerDefaults.AuthenticationScheme)
    .AddMicrosoftIdentityWebApi(builder.Configuration.GetSection("EntraId"));

builder.Services.AddAuthorization(o =>
    o.AddPolicy("SameTenant", p => p.AddRequirements(new TenantIsolationRequirement())));

builder.Services.AddAhsCommon(builder.Configuration);      // from AHS.Common
builder.Services.AddColdChainInfrastructure(builder.Configuration);

var app = builder.Build();

app.UseAuthentication();
app.UseAuthorization();
app.UseMiddleware<TenantResolutionMiddleware>();

app.MapGroup("/api/shipments").MapShipmentEndpoints().RequireAuthorization("SameTenant");
app.MapGroup("/api/oracle").MapOracleEndpoints().RequireAuthorization("SameTenant");
app.MapGet("/health", () => Results.Ok(new { status = "healthy", cell = "ColdChain" }))
   .AllowAnonymous();

app.Run();
public partial class Program { }
```
